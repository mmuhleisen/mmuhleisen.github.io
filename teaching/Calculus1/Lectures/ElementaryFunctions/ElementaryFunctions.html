Elementary functions

Exposition

The vast majority of functions we will talk about can be constructed using the ones listed below as building blocks.

1

Fix a real number α∈ℝ. Among the most common algebraic, exponential, and trigonometric functions are those listed below, along with their domains and codomains.

(In this class, log⁡(x) will always refer to the natural logarithm; the notation ln⁡(x) will be avoided. There is no reason other than personal preference. Also, ex is often alternatively denoted exp⁡(x), both in this class and elsewhere.)

Exposition

In fact, we have a name for functions which can be obtained recursively from the ones listed above using a combination of arithmetic operations and composition.

Definition

2

Let U=(a,b) be an open interval, and let f:U→ℝ be a function. We say that f is elementary if it is one of the functions listed in the previous example, or if it is a sum, product, or composite of elementary functions.

Exposition

The purpose of this definition is to give a name to the set of functions which you can simply “write down.” The following example illustrates this idea.

Example

3

The expression

x3+π⁢x2−e⁢x+2x2+1
defines an elementary function ℝ→ℝ, as do eeeex and sin⁡(π2−x)=cos⁡(x).

Exposition

It’s true that elementary functions are those functions that you can just “write down,” but not everything you can just “write down” is a function! Being explicit about domains and codomains of functions helps us avoid the following pitfall.

Warning

4

There are expressions such as log⁡(1+x−ex) which look like elementary functions, but which may not be defined on a given interval. In this example, log is a function ℝ+→ℝ defined only on the positive real numbers while the maximum value of the function f:ℝ→ℝ given by f⁢(x)=1+x−ex is zero. Thus, neither can the functions f and log be composed, nor can any of their restrictions be composed. If one had taken care to check the domains and codomains of log⁡(x) and f⁢(x), one would never even have written down the expression log⁡(1+x−ex)!

In closing, we’ll mention that elementary functions are very special. Even though there are (infinitely) many elementary functions, and even though these elementary functions can get quite complicated (despite their name!), the vast majority of abstract functions are not elementary. Many more examples will arise when we discuss integration.

Example

5

The following functions are not elementary.

  1. The floor function ⌊−⌋:ℝ→ℝ whose value at x is the greatest integer which is less than or equal to x. For example, ⌊4⌋=4=⌊4.1⌋ and ⌊3.9⌋=3=⌊3.5⌋.

  2. The Dirichlet function δ:ℝ→ℝ whose value at all rational numbers is 1, and whose value at all irrational numbers is 0. For instance, δ⁢(π)=0 although
